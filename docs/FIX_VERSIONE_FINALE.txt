CivicVois - versione finale Netlify + Supabase

Correzioni principali:
- lettura segnalazioni senza join embedded PostgREST, quindi meno errori di schema cache;
- gestione chiavi Supabase sb_publishable_* più sicura lato browser;
- retry automatico sui piccoli errori di rete/cold start;
- login e registrazione con gestione errori più chiara;
- creazione/recupero profilo più robusta;
- file supabase/fix-existing-project.sql incluso per aggiornare il trigger del profilo se necessario.

Dopo il deploy:
1. apri Netlify -> Deploys -> Deploy manually;
2. carica questo ZIP;
3. apri il sito in incognito o fai CMD+SHIFT+R;
4. se la registrazione crea l'utente ma non entra subito, controlla in Supabase: Authentication -> Providers -> Email -> Confirm email. Per test immediati puoi disattivare la conferma email.

Non mettere mai nel frontend una chiave sb_secret_*.
