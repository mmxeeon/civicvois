CIVICVOIS - DEPLOY CORRETTO

Questa versione usa una Netlify Function proxy:
  netlify/functions/civicvois-api.js

Per questo NON basta il drag-and-drop dello ZIP come se fosse solo HTML/CSS/JS.
Devi deployarla tramite uno di questi metodi:

METODO CONSIGLIATO: GITHUB
1. Crea un repository GitHub.
2. Carica tutto il contenuto di questa cartella nel repository.
3. Su Netlify fai Add new project -> Import from Git.
4. Seleziona il repository.
5. Lascia publish directory: .
6. Lascia build command vuoto.
7. Deploy.
8. In Netlify controlla che in Functions compaia civicvois-api.

METODO ALTERNATIVO: NETLIFY CLI
1. Apri il Terminale nella cartella del progetto.
2. netlify login
3. netlify link
4. netlify deploy --prod
